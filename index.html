Quick Start

1 Import: Drop CSV, CSV.GZ, JSON, Parquet files
2 Autocomplete: Type for suggestions, Tab to accept
3 Run: Ctrl+Enter to execute
4 Export: Copy or download CSV

Supported Files

CSV CSV.GZ JSON / JSONL Parquet Excel

CSV.GZ files are decompressed in JavaScript before loading into DuckDB for maximum compatibility.

Essential Queries

SELECT * FROM table_name LIMIT 100;
SELECT COUNT(*), AVG(price) FROM sales GROUP BY category;
COPY (SELECT * FROM table) TO 'output.csv' (HEADER);

Advanced Analytics

Window Functions

SELECT product, sales, SUM(sales) OVER (ORDER BY date) as running_total, RANK() OVER (PARTITION BY category ORDER BY sales DESC) as rank FROM sales;

Common Table Expressions (CTEs)

WITH top_products AS ( SELECT product, SUM(quantity) as total FROM orders GROUP BY product ORDER BY total DESC LIMIT 10 ) SELECT * FROM top_products;

ASOF Join (Time-series)

SELECT trades.*, prices.price FROM trades ASOF JOIN prices ON trades.symbol = prices.symbol AND trades.time >= prices.time;

Pro Tips

  • SUMMARIZE table_name - Instant statistics on any table
  • DESCRIBE table_name - See column types
  • QUALIFY - Filter window functions without subqueries
  • FILTER (WHERE condition) - Conditional aggregates
